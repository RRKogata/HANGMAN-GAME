# HANGMAN-GAME

A simple Flask-based Hangman game with categories, difficulty levels, hints, and a keyboard UI.

## Run locally

```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\\Scripts\\activate
pip install -r requirements.txt
python app.py
```

Then open http://127.0.0.1:5000/

## GitHub publish

```bash
git remote add origin https://github.com/<your-github-username>/HANGMAN-GAME.git
git branch -M main
git push -u origin main
```
